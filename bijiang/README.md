# 碧江影游 · 非遗主题互动影游

> 一个以"保护非遗"为主题的村落沉浸式互动影游 H5 应用

## 🎮 项目简介

碧江影游是以中国历史文化名村——顺德北滘碧江村为场景的互动影游原型。玩家以不同身份（苏家子弟、京城公主、广雅学子、账房先生、后世寻访者）走进碧江古村，通过9个站点的打卡、剧情观看、解谜探索，发现散落在古建中的非遗瑰宝，了解其工艺、故事与保护现状，最终获得"非遗守护者"证书。

## ✨ 核心特色

- **9站沉浸式游线**：村心街 → 职方第 → 慕堂祠 → 金楼 → 五间祠 → 振响楼 → 亦渔遗塾 → 德云桥 → 金楼后花园
- **5种身份代入**：不同视角看到不同故事
- **4类解谜玩法**：找物谜题、密码谜题、组合谜题、排序谜题
- **15件非遗瑰宝**：砖雕、灰塑、金漆木雕、满洲窗、蚝壳墙、石湾陶塑等，每件含工艺/故事/保护三段深度内容
- **4级稀有度**：普通 / 珍贵 / 珍稀 / 国宝
- **4个关键抉择**：影响最终结局（守护者/见证者/办学者/守成者/寻访者）
- **非遗图鉴**：点击任意收集的非遗查看完整详情
- **进度持久化**：localStorage 自动保存，刷新不丢失

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript（单文件，无任何依赖）
- 移动端优先响应式设计（最大宽度 460px）
- 支持扫码解锁（`wx.scanCode` 接口预留）

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可体验完整流程：

1. 选择身份 → 确认出发
2. 从 S1 村心街入口开始逐站打卡
3. 每个站点：观看剧情 → 破解谜题 → 收集非遗 → 做出选择
4. 集齐9枚记忆碎片后合成"碧江非遗守护者证书"

## 🔄 可复制模式

本项目采用"村落人物互动影游模板"设计，可快速复制到其他古村落：

1. 替换 `STATIONS` 数组为目标村落的站点数据
2. 替换 `ITEMS` 对象为当地非遗/文保单位信息
3. 替换 `CHOICES` 对象为符合当地人物故事的抉择
4. 替换 `PUZZLES` 对象为匹配的谜题
5. 调整 `IDENTITIES` 为适配的人物身份
